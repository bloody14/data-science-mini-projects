# 🚀 Data Science Foundations Projects

A collection of hands-on Data Science mini-projects built using **Python, NumPy, Pandas, and Matplotlib** in **Google Colab**.

This repository documents my learning journey through the foundational concepts of Data Science by working on practical projects involving numerical computing, data manipulation, analysis, and visualization.

---

## 📌 About This Repository

The goal of this repository is to strengthen core Data Science skills through project-based learning. Each project focuses on a specific concept and demonstrates how popular Python libraries are used to solve real-world problems.

All notebooks were developed and executed using **Google Colab**.

---

## 🛠️ Technologies Used

* Python 3
* NumPy

* Matplotlib

## 📂 Projects Completed

### 1️⃣ Student Grade Analyzer (NumPy)

**Objective:** Analyze student exam scores using NumPy arrays.

#### Key Concepts

* Array creation and manipulation
* Mean, Median, Standard Deviation
* Boolean Masking
* Broadcasting
* Reshaping Arrays

#### Features

* Generate student scores
* Calculate statistical measures
* Identify distinction and failing students
* Normalize scores
* Subject-wise performance analysis

---

### 2️⃣ Grayscale Image Manipulator (NumPy + Matplotlib)

**Objective:** Understand images as matrices and manipulate them using NumPy.

#### Key Concepts

* Image arrays
* Grayscale conversion
* Brightness adjustment
* Image flipping
* Image cropping

#### Features

* Convert RGB image to grayscale
* Increase/decrease brightness
* Horizontal and vertical flip
* Crop selected regions
* Visualize image transformations

---

### 3️⃣ IPL Match Data Explorer (Pandas)

**Objective:** Explore IPL match datasets using Pandas.

#### Key Concepts

* Reading CSV files
* Data inspection
* Filtering and sorting
* GroupBy operations
* Handling missing values

#### Features

* Season-wise match analysis
* Team performance insights
* Venue-based filtering
* Player of the Match statistics
* Missing value detection

---

### 4️⃣ Sales Performance Dashboard (Pandas)

**Objective:** Analyze business sales data using aggregation and grouping techniques.

#### Key Concepts

* GroupBy
* Aggregations
* Pivot Tables
* Lambda Functions
* Feature Engineering

#### Features

* Region-wise sales analysis
* Category-wise profit tracking
* Profit margin calculation
* Pivot table creation
* Business performance insights

---

### 5️⃣ E-Commerce Order Intelligence (Pandas)

**Objective:** Combine multiple datasets and generate business insights.

#### Key Concepts

* Data Merging
* Joins
* Concatenation
* Reshaping Data
* Date-Time Operations

#### Features

* Customer revenue analysis
* Product performance analysis
* Order trend exploration
* Data transformation using Melt and Pivot
* Customer behavior insights
